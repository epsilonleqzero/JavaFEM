Description and Features

Written and developed by Ted Kwan

Welcome to my website! It is operational, but new features are being added all of the time. This website serves as a sandbox for programming in Java EE and C++ on Ubuntu. I hope you enjoy your stay, and will try out the features on the left.

There are currently three working features. These are the utility to solve the Poisson equation on a square, then plot the result, plot a 2D phase plane, and plot a 3D phase plane.. All of the numerical calculations are done when the feature is clicked, and no graphics have been pre-saved on them. This allows for testing of the subroutines and will soon allow for modification of the parameters.

This website was built using Java EE 7 and JSF 2.2. The features at the left of the page are examples of numerical programming in C++ (using JNI to link between Java and C++). The linear algebra is all done with the Armadillo library which is a wrapper for BLAS and LAPACK. This was done to ensure speed up the processing time, since Java is not known for computational mathematics and speed. Many of the subroutines were translated from MATLAB, and some were written originally for this website. The github repository linked on the left contains the code for this site, and contains many other code samples.
